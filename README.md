# 나스닥 급등주 스크리너 Centro Pro v2.0

실전형으로 다시 손본 버전입니다.

## 이번 버전 핵심
- **티커 검색 정상화**
- **자동 후보 발굴 스캔**
- **프리셋 스캔**
  - 전체 후보
  - 프리마켓 강세
  - 뉴스 촉매
  - 개장 돌파
  - 눌림목 반등
- **점수 기반 정렬**
- **선택 종목 15초 라이브 갱신**
- **이중 볼린저밴드 차트**
- **진입가 / 손절가 / 목표가**
- **워치리스트 저장**
- **DEMO 모드 지원**

## 왜 이전 버전이 검색이 약했나
이전 버전은 사실상 워치리스트 스캐너에 가까웠습니다.
이번 버전은 `/api/search`와 `/api/scan`을 분리해,
- 티커 검색
- 종목 즉시 추가
- 후보군 자동 스캔
이 실제로 동작하도록 다시 구성했습니다.

## 실행
```bash
npm install
npm run dev
```

## .env.local 예시
```env
FINNHUB_API_KEY=여기에_키
GROQ_API_KEY=여기에_키
NEXT_PUBLIC_DEMO_MODE=false
```

API 키 없이 화면 테스트만 하려면:
```env
NEXT_PUBLIC_DEMO_MODE=true
```

## 현실적인 한계
이 앱은 **후보 발굴기 + 압축기**로 설계했습니다.
무료 Finnhub 기준으로는 전 나스닥 종목을 초단위로 훑는 수준의 기관형 스캐너는 어렵습니다.

그래도 아래에는 꽤 쓸 만하게 만들었습니다.
- 후보 압축
- 뉴스 촉매 확인
- 기술적 자리 확인
- 워치리스트 관리
- 실시간 재확인

최종 체결 판단은 TradingView, 브로커 호가, 체결강도와 같이 보시는 것을 권장합니다.


## v2.1 검토/수정 사항
- **Finnhub 무료 호출 한도 대응**
  - 자동 스캔 시 기본 최대 개수를 줄이고 서버 캐시를 추가했습니다.
  - 실데이터 모드에서 무리한 전수 스캔으로 막히는 문제를 완화했습니다.
- **워치리스트 실전 스캔 추가**
  - 로컬 워치리스트만 따로 스캔 가능
- **검색 결과 클릭 개선**
  - 검색 결과에서 바로 추가/불러오기 가능
- **오류 메시지 보강**
  - 호출 제한/심볼 오류 시 안내 강화

## 권장 운용
무료 API 기준으로는 아래 방식이 가장 현실적입니다.
1. 검색으로 후보 추가
2. 워치리스트 6~10종목 위주로 스캔
3. 상위 점수 종목만 라이브 확인
4. 마지막 진입은 차트/호가로 재확인
